BEAR PEAR channel

Registering the channel: 
pear channel-discover pear.bear-project.net
Listing available packages: 
pear remote-list -c bear
Installing a package: 
pear install bear/package_name
Installing a specific version/stability: 
pear install bear/package_name-1.0.0
pear install bear/package_name-beta
Receiving updates via a feed: 
http://pear.bear-project.net/feed.xml
